## ğŸ¯ Example: Quick Roll

```
/roll_char character:Raven attribute:Dexterity skill:Firearms difficulty:2
```

Herald automatically:
- Looks up your character "Raven"
- Gets their Dexterity and Firearms ratings
- Rolls the combined pool
- Checks for successes vs. difficulty 2
- Displays results with Edge/Desperation if applicable
