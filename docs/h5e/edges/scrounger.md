Find anything. Anywhere. Always.

Scrounger represents a Hunter's supernatural ability to locate resources, discover hidden items, and improvise solutions from nothing. You always find what you need, no matter how unlikely.

---

## Overview

### What is Scrounger?

Scrounger is an **Aptitude Edge** that grants supernatural luck and resourcefulness. When you need something - equipment, information, shelter - you find it through impossible coincidences and supernatural providence.

The Edge operates through **supernatural providence**: the universe provides what you need when you need it. Luck, intuition, and impossible coincidences conspire to supply you with resources.

### Who Should Take This Edge?

**Ideal For:**
- **Survivors** - Find resources in any situation
- **Improvisers** - MacGyver-style problem solving
- **Poor Hunters** - No money, all luck
- **Innocents** - Stumble into solutions
- **Support characters** - Supply the team

**Consider Carefully If:**
- You have wealth/resources already (Arsenal, Status)
- You prefer planned approaches
- Storyteller doesn't like coincidence mechanics
- You want reliable/predictable resources

### Key Strengths

- **Always equipped** - Find what you need
- **Improvisation** - Create solutions from junk
- **Survival** - Never without basics
- **Flexibility** - Adapt to any situation
- **Self-sufficient** - Don't need money/connections

### Limitations

- **Unpredictable** - What you find varies
- **Storyteller-dependent** - GM determines availability
- **Improvised quality** - Items may be suboptimal
- **No control** - Find what's there, not what you want exactly
- **Requires scavenging** - Must search/look

## All Perks

### Find It

**What It Does:**
Locate any common item or resource in your environment through supernatural luck. Add your Desperation rating to Survival or Investigation rolls when searching.

**Mechanical Effect:**
- Searching: Base pool + Desperation
- Find common items in unlikely places
- Success = discover what you need

**Common Items:**
- Food, water, shelter
- Basic tools (crowbar, flashlight, rope)
- First aid supplies
- Common weapons (pipes, knives)
- Useful junk

**When To Use:**
- **Need supplies** - Out of resources
- **Improvise solutions** - Need specific tool
- **Survival situations** - Lost/stranded
- **Quick equipment** - No time to shop

**Strategy Notes:**
- Foundation perk for Scrounger
- Works in any environment
- Better with high Desperation
- Pairs with Survival skill

**Example:**
```
Need crowbar to pry open door
Find It: Search nearby abandoned building
Roll: Wits 3 + Survival 2 + Desperation 3 = 8 dice
/roll pool:8 difficulty:2 comment:Finding crowbar
Success: Discover crowbar in maintenance closet
Problem solved through providence
```

**Herald Commands:**
```
/edge edge:Scrounger
/perks edge:Scrounger perk:Find It
```

### Improvise

**What It Does:**
Create functional tools or weapons from junk and scavenged materials. Build what you need from what's available.

**Applications:**
- **Weapons** - Improvised clubs, spears, explosives
- **Tools** - Lockpicks, pry bars, grappling hooks
- **Equipment** - Radios, tracking devices, traps
- **Medical** - First aid from household items

**When To Use:**
- **Weapon loss** - Replace broken/lost gear
- **Trapped** - Create escape tools
- **Resource denial** - Enemy took your equipment
- **Specialized needs** - Create one-off custom tools

**Strategy Notes:**
- Requires Craft skill
- Items are improvised quality (may break)
- Creative problem solving
- Storyteller determines feasibility

**Example:**
```
Lost weapons, need to fight ghouls
Improvise: Create weapons from warehouse materials
- Pipe becomes club
- Chain becomes flail
- Lighter + aerosol = flamethrower
Roll Craft: Success
Improvised arsenal ready for combat
```

**Herald Commands:**
```
/perks edge:Scrounger perk:Improvise
```

### Unlikely Find

**What It Does:**
Discover rare or unusual items through impossible luck. Find things that shouldn't be there but somehow are.

**Unlikely Items:**
- Rare weapons in junkyards
- Holy water in abandoned churches
- Medical supplies in condemned buildings
- Silver bullets in pawnshops
- Occult texts in garage sales

**When To Use:**
- **Specific threats** - Need silver for werewolves
- **Rare resources** - Find unlikely items
- **Plot convenience** - Storyteller enables discovery
- **Desperate situations** - Impossible luck activates

**Strategy Notes:**
- Most powerful Scrounger perk
- Storyteller-dependent heavily
- Can find plot-relevant items
- "Coincidence" as superpower

**Example:**
```
Fighting vampire, need wooden stakes urgently
Unlikely Find: Search abandoned church
Roll amazing success
Discover box of old wooden fence posts (perfect stakes)
Left by previous vampire hunters decades ago
Providence provides
```

**Herald Commands:**
```
/perks edge:Scrounger perk:Unlikely Find
```

### Salvage

**What It Does:**
Recover and repair damaged equipment. Fix broken weapons, restore damaged gear, and make things work despite damage.

**Applications:**
- **Repair weapons** - Fix broken firearms
- **Restore equipment** - Damaged electronics work
- **Vehicle repair** - Get cars running
- **Structural fixes** - Temporary building repairs

**When To Use:**
- **Equipment damage** - Gear breaks during mission
- **Post-combat** - Repair after fights
- **Resource scarcity** - Can't replace, must fix
- **Battlefield scavenging** - Take enemy equipment

**Strategy Notes:**
- Requires appropriate skills (Craft, Technology)
- Extends equipment lifespan
- Enables use of enemy gear
- Pairs with Improvise

**Example:**
```
Rifle jammed during vampire fight
Salvage: Quick field repair
Roll Craft + Desperation
Success: Rifle functional again
Continue fighting
Post-combat: Properly repair later
```

**Herald Commands:**
```
/perks edge:Scrounger perk:Salvage
```

---

### Always Prepared

**What It Does:**
Retroactively have small, reasonable items that you "picked up earlier." Produce items from pockets as needed.

**Mechanical Effect:**
- Spend Willpower point
- Declare you have small common item
- Produce it from pocket/bag
- Must be reasonable/small

**Reasonable Items:**
- Lighter, knife, flashlight
- Phone charger, batteries
- First aid basics
- Keys, lockpicks
- Rope, duct tape

**When To Use:**
- **Forgot supplies** - Realize you need something
- **Unexpected situations** - Didn't plan for this
- **Quick solutions** - Need item immediately
- **Backup preparation** - Always have essentials

**Strategy Notes:**
- Costs Willpower (limited use)
- Must be plausible (no guns from pockets)
- Great for "prepared" characters
- Storyteller veto on unreasonable items

**Example:**
```
Need to pick lock on vampire's lair door
Always Prepared: "I picked up lockpicks earlier"
Spend 1 Willpower
Produce lockpicks from pocket
Storyteller agrees reasonable
Proceed with lock picking
```

**Herald Commands:**
```
/perks edge:Scrounger perk:Always Prepared
```

## Strategy Guide

### Early Game (Characters 0-20 XP)

**Starting with Scrounger:**
- **Focus:** Basic resource finding
- **Best Perks:** Find It, Improvise
- **Build:** Wits 3, Survival 3, Craft 2
- **Tactics:** Scavenge before missions, improvise solutions

**Recommended Build:**
- Wits 3, Intelligence 2, Dexterity 3
- Survival 3, Craft 3, Awareness 2
- Find It first (basic scavenging)
- Improvise second (create tools)

**Scavenger Pattern:**
- Search area before operations
- Find basic supplies
- Improvise missing equipment
- Share finds with team

### Mid Game (Characters 21-50 XP)

**Developing Scrounger:**
- **Add Perks:** Salvage, Always Prepared
- **Expand Role:** Team supplier and improviser
- **Build skills:** Raise Survival and Craft
- **Resource independence:** Never without essentials

**Advanced Tactics:**
- Comprehensive pre-mission scavenging
- Battlefield salvage during fights
- Always Prepared for contingencies
- Supply team from found resources

**Progression:**
- Raise Survival to 4-5 (best scavenging)
- Raise Craft to 4-5 (best improvisation)
- Raise Awareness (spot resources)
- Second Edge: Arsenal (procurement + scavenging), Senses (find better)

### Late Game (Characters 51+ XP)

**Mastering Scrounger:**
- **All Perks:** Find anything, always
- **Unlikely Find:** Discover impossible items
- **Perfect preparation:** Always have what's needed
- **Team supplier:** Keep everyone equipped

**End-game Tactics:**
- Unlikely Find for rare/plot items
- Comprehensive salvage operations
- Always Prepared for any situation
- Never without resources

**Ultimate Build:**
- Wits 4, Intelligence 3, Dexterity 4
- Survival 5, Craft 5, Awareness 4
- All Scrounger perks
- Second Edge: Senses (spot everything)
- Ultimate improviser and finder

### Edge Synergies

**Works Great With:**
- **Innocent Creed** - Stumble into resources providentially
- **Defender Creed** - Find resources to protect others
- **Senses Edge** - See hidden resources
- **High Wits** - Spot opportunities faster
- **Craft/Survival skills** - Maximize finding/making

**Conflicts With:**
- **Arsenal/Status** - Already have resources (overlap)
- **Wealthy concepts** - Why scavenge if rich?

**Neutral:**
- **Library, Contacts, Fleet** - Different focus but compatible

### Using Scrounger in Play

Scrounger operates through searching and providence:

1. **Identify need** - What do you need?
2. **Search environment** - Roll Survival/Investigation + Desperation
3. **Find result** - Storyteller determines what you find
4. **Use resource** - Apply solution

**Example:**
```
Need first aid supplies for wounded ally
Environment: Abandoned office building
Find It active: Roll Survival
/roll pool:8 difficulty:2 comment:Finding first aid (Wits 3 + Survival 2 + Desperation 3)
Result: 5 successes
Storyteller: "You discover a fully-stocked first aid kit in old break room"
Treat ally's wounds
Providence provides
```
